# CMSC434-Impressionist

## Instructions
1. Clone or download repo
2. Extract (if downloaded as zip)
3. Import project into Android Studio
4. Run on device or emulator

## Features

1. Download images
2. Load image
3. Choose from 3 paint brushes - circle, square, line
4. Clear painting
5. Save image
6. Invert painting
